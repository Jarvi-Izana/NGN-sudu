  • /sudu6770/

    • GET: Brief introduction about SUDU

  • /sudu6770/register/

    • POST: register a user, after login, the login status is True, invisible by client.

    SAMPLE: response = client.post('/sudu6770/register/', {'user_name': 'woo', 'email_addr': 'woo@columbia.edu', 'password':'woo', 'token':'12345', 'project_name': 'sudu'})

  • /sudu6770/login/

    • ALL THE REST COMMANDS WILL TAKE INTO EFFECT AFTER LOGIN. IP ADN ONLINE COOKIE WILL BE TRACED.

    POST: login a user with JSON(user_name,password) or JSON(email_addr, password).

    SAMPLE: response = client.post('/sudu6770/login/', {'user_name':'woo', 'password':'woo'});

  • /sudu6770/logout/

    • FOR SECURITY PURPOSE, PLEASE LOGOUT EVERY TIME

    POST: logout a user. The user online status will be set False.

    SAMPLE: response = client.post('/sudu6770/logout/', {'email_addr':'gui@gmail.com', 'password':'gui'});

  • /sudu6770/projeccts/

    • GET: test page, will show the client ip

    POST: show group member, prerequisite: login. Show the members within same projects

    SAMPLE: response = client.post('/sudu6770/projects/', {'email_addr': 'peng@gmail.com', 'project_name': 'sudu'})

  • sudu6770/projects/finish/

    • POST: finish a task, set project_status=True. This item is used to estimated the total procedure of the project.

    SAMPLE: response = client.post('/sudu6770/projects/finish', {'email_addr':'gui@gmail.com', 'project_name':'sudu'});

  • sudu6770/projects/add/

    • POST: add yourself a task or invite others.

    SAMPLE: response = client.post('/sudu6770/projects/add/', {'email_addr': 'woo@columbia.edu', 'project_name':'sudu'});

  • /sudu6770/projects/quit/

    • POST: quit from a project.

    SAMPLE: response = client.post('/sudu6770/projects/quit/', {'email_addr': 'woo@columbia.edu', 'project_name':'sudu'});

  • /sudu6770/personal/

    • POST: query the user's all projects.

    SAMPLE: response = client.post('/sudu6770/personal/', {'email_addr' : 'xi@columbia.edu'})

  • /sudu6770/unregister/

    • POST: unregister a user. Delete permanently

    SAMPLE: response = client.post('/sudu6770/unregister/', {'email_addr': 'woo@columbia.edu', 'password':'woo'});
